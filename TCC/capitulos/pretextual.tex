\resumo{%
Este trabalho propõe o desenvolvimento de um sistema de monitoramento em tempo real para a irrigação agrícola, integrando tecnologias modernas com o método científico da Evapotranspiração da Cultura (ETc). O sistema visa fornecer aos agricultores informações precisas sobre as necessidades hídricas das culturas, utilizando o protocolo MQTT para comunicação entre dispositivos e a aplicação de IoT na agricultura. Espera-se que o sistema contribua para a gestão eficiente da água na agricultura, promovendo a sustentabilidade e a produtividade das plantações.
}
\palavraschave{Monitoramento em tempo real. Agricultura de precisão. Evapotranspiração da Cultura (ETc). Protocolo MQTT. Internet das Coisas (IoT).}


% --------------------------------------------------
% Keywords e abstract
% --------------------------------------------------
\abstract{%
This work proposes the development of a real-time monitoring system for agricultural irrigation, integrating modern technologies with the scientific method of Crop Evapotranspiration (ETc). The system aims to provide farmers with accurate information about crop water needs, using the MQTT protocol for device communication and applying IoT in agriculture. It is expected that the system will contribute to efficient water management in agriculture, promoting sustainability and productivity in crops.
}
\keywords{Real-time monitoring. Precision agriculture. Crop Evapotranspiration (ETc). MQTT protocol. Internet of Things (IoT).}
